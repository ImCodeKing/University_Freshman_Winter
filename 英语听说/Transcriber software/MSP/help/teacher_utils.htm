教学增益集 说明　

        当有数个片断要播放时，用影音剪辑软体是非常耗时的，而且也无法剪后调整。
所以此教学增益功能集，以加入 tag 在时间标记后的方式，快速方便满足此一需求。
 


note 放大图


　

注意：需将右键选单中之 " open control mode " 　开启，各略过功能才会作用。

新增 tag ：
                将滑鼠移至所要插入的行，按右键选单加入。
                另一种方式是自己打字进去。但应打在时间标记之后方。编辑结束后按右键选单之 refresh选项，完成更新。
移除 tag ：
                 直接 delete. 并按refresh选项。
